animat.io

A Modern CSS Animations / Effects Library

Getting Started

Usage

You can apply any animation by just by calling .animation method and defining the animation type to any element. 

<script type="text/javascript">

  animation.animatio(selector).animate('fadeIn');

</script>
Configure

Effects are completely customizable. Default configuration settings:

  • duration Default: '1s'
  • timing Default: 'ease'
  • iterationCount Default: 1
  • direction Default: 'normal'
  • delay Default: '0s'
  • fillMode Default: 'forwards'
You can read more about each configurable property here - Official W3C Document. 
<script type="text/javascript">

  animatio(selector).animate('pulse', {
    delay: '1s',
    duration: '2s',
    iterationCount: 'infinite'
  });

</script>
Callback

You can add an optional third parameter to define a callback function that will be executed when the animation ends. 

<script type="text/javascript">

  animatio(selector).animate('fadeIn', function(){
    animatio(this).animate('fadeOut');
  });

</script>
Customize

There are predefined animations that are ready to use (listed below). However, you can also create your own animation by setting it as a string in config.rule.

{browser} is a place-holder that we will be updated with the correct browser prefix once the rule has been appended to the document head. 

<script type="text/javascript">

  animatio(selector).animate('moveLeft', {
    rule: "0% { {browser}transform: translateX(0);}, 100% { {browser}transform: translateX(100);}"
  });

</script>
